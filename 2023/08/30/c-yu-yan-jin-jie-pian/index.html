结构体数组

定义格式和一般定义结构差不多，只是变量名为数组。每一个数组都有结构体的属性。但属性值要分别定义。以下用循环定义每一个结构体数组的属性值。

#include <stdio.h>

struct student
{
    int num;
    char sex;
} a[2];

void main()
{
    int i;
    for (i = 0; i < 2; i++)
    {
        a[i].num = 10;
        a[i].sex = 'a';
        printf("%d", a[i].num);
        printf("%c\n", a[i].sex);
    }
}

10a

计算结构体所占用的内存

结构体的内存存储讲究对齐法则。所有属性所占的内存向占最大内存的那个属性对齐，例如计算以下结构体所占内存，char类型占1个字节，而int类型占4个字节，所以char类型向int类型对齐，则以下结构体所占内存为8。

#include <stdio.h>

struct student
{
    int num;
    char sex;
};

void main()
{
    printf("%d", sizeof(struct student));
}`

结果为8。

以上为student结构体在内存的情况，0-3为整型类型num的存储空间，4-7为char类型sex的存储空间。

共用体的简单介绍

上述结构体所占内存可见非常废内存，char类型占用1个字节就占一个字节，为什么占用4个字节，所以共用体就比较节约内存了。共用体内存看所占内存最大的那个类型。其他类型共用它的内存。

共用体的定义和结构体的定义差不多，结构体定义所用的关键字为struct，共用体的定义所用关键字为union。所以以下仅对共用体所占内存进行描述。

#include <stdio.h>

union student
{
    char uni;
    char sex;
} a;

void main()
{
    a.uni = 'b';
    a.sex = 'a';
    printf("%c\n", a.uni);
    printf("%c\n", a.sex);
    printf("%d", sizeof(union student));
}

以上代码的结果为

a

a

1。

以上为共用体的占用内存情况，uni和sex，因为uni和sex都仅占一个内存，所以赋值进行的覆盖，先赋值b，如何赋值a，但共用一个内存，所以a覆盖了b。

枚举的介绍

之前介绍了结构体和共用体，结构体的关键字为struct，共用体的关键字为union，而枚举定义的关键字为enum。枚举类型简化了#define定义多个值的操作

#include <stdio.h>
enum week
{
    a,
    b,
    c,
    d
} p;//week为枚举名，p为枚举变量
void main()
{
    printf("%d", a);
    printf("%d", b);
    printf("%d", c);
    printf("%d", d);
}

结果为0123。可以认为枚举中的变量值从0开始赋值。

枚举的简单应用

#include <stdio.h>
enum week//枚举名
{
    first,
    two,
    three,
    four
} p;//枚举的变量名p
void main()
{
        int k = 0;
        p = (enum week)k;//对枚举变量进行赋值的变量也要是枚举类型，所以这里要进行强制类型转换
        switch (p)
        {
        case first:
            printf("first");
            break;
        case two:
            printf("two");
            break;
        case three:
            printf("three");
            break;
        case four:
            printf("four");
            break;

        default:
            break;
        }
}

以上输出的结果为first。可以认为枚举变量中存储了

枚举所占空间

C++标准文档中是这样说明的：“枚举类型的尺寸是以能够容纳最大枚举子的值的整数的尺寸”，同时标准中也说名了：“枚举类型中的枚举子的值必须要能够用一个int类型表述”。

#include <stdio.h>

    a
} p;
void main()

    printf("%d", sizeof(enum week));

无论怎么添加枚举中的成员，输出的结果都为4个字节即一个整型类型的大小

指针变量的介绍

变量有int类型 char类型 doubke类型等等,指针变量也有同样的类型,不过指针的值是一个变量的地址,,实际上的是指针指向的是变量的地址

#include <stdio.h>
void main()
{
    int *a; // 定义了一个整型类型的指针
    int b = 8;
    a = &b;             // 指针变量a指向整型变量b的地址
    printf("%d\n", *a); // 打印指针a指向变量的值
    printf("%d\n", a);  // 打印a指向的地址
    printf("%d\n", &a); // 打印指针a的地址
    printf("%d", &b);   // 打印变量b的地址
}

void类型的指针

C语言中的void类型，代表任意类型，而不是空的意思，而是说它的类型是未知的，是还没指定的。
void * 是void类型的指针。void类型的指针的含义是：这是一个指针变量，该指针指向一个
void类型的数。void类型的数就是说这个数有可能是int，也有可能是float，也有可能是个结构体，哪种类型都有可能，只是我当前不知道。

#include <stdio.h>

void main()
{
    int a = 5;
    char b = 'a';
    void *p;
    p = &a;                    // p指向这个地址
    printf("%d\n", *(int *)p); // 因为这个数据是int类型的，所以要强制转换成对应类型的指针
    p = &b;
    printf("%c\n", *(char *)p); // 因为这个数据是char类型的，所以要强制转换成对应类型的指针
}

输出结果为

5
a

指针函数（和函数指针区别）

说到函数指针就不得不提到指针函数了，指针函数实质上是一个函数不过返回的是一个指针。

#include <stdio.h>
void main()
{
    int *f(int a, int b);
    int *p;      // 定义一个指针用来接收返回的地址
    p = f(2, 3); // 指针p接收指针函数的返回的指针
    printf("%d", *p);
}

int *f(int a, int b)
{
    int *max;
    if (a > b)
    {
        max = &a;
    }
    else
    {
        max = &b;
    }
    return max; // 返回一个指针，这个指针指向的是最大值的地址
}

数组指针

数组指针同理，本质上也是一个指针不过指向的是一个数组，直接上代码感觉下面的代码已经很详细了

#include <stdio.h>
void main()
{

    int b[2][3] = {1, 2, 3, 4, 5, 6};
    int a[2] = {1, 2};
    int(*p)[3]; // 定义了一个二维数组的指针，注意要[3]为该数组指针的边界值
    p = b;      // 二维数组指针指向b数组
    int(*q);    // 定义了一个一维数组的指针
    q = a;      // 一维数组指针指向a数组
    printf("%d", *(q + 1));
    printf("%d", *(*(p + 1) + 1)); // 注意如果没有给出数组指针p的[3]这个边界值*(p+1)会报错说没有给出指针的边界值
}

输出的结果为

链表的介绍

理解为一种思想，用结构体指针来实现的，结构体中的成员包括数据和结构体指针，结构体中的结构体指针指向的是下个结构体的地址，即用结构体中的结构体指针来链接各个结构体，整体称谓链表，那么有了链表就需要一个头指针来输出他，可以理解链表是多米诺骨牌，牌已经摆好了，只用推一下这个链表就展现到你眼前了，而这个推一下的动作就是头指针，

第一种静态链表的介绍与实现

#include <stdio.h>

typedef struct student
{
    int score;
    struct student *next; // 创建一个节点
} p;                      // 重新定义了struct student这个类型

void main()
{
    p a, b, c, d, *head;
    a.score = 10; // 给结构体变量赋值
    b.score = 20;
    c.score = 30;
    d.score = 40;

    head = &a;     // 头指针保存a的地址
    a.next = &b;   // a的节点保存结构体变量b的地址
    b.next = &c;   // b的节点保存结构体变量c的地址
    c.next = &d;   // c的节点保存结构体变量d的地址
    d.next = NULL; // 给最后一个节点写个内容 用于判断节点是否已经遍历完成

    while (head != NULL) // 头指针如果没有保存最后一个节点的内容 就一直遍历
    {
        printf("%d  ", head->score); // 遍历score
        head = head->next;           // 把结构体中的next指针赋值给head结构体指针，则head指针指向的是下一个节点的地址。
    }
}

输出结果为

10 20 30 40

第二种动态链表的介绍与实现

静态链表是提前知道了有几个结构体变量，而实际上我们并不知道我们需要10个或者100个结构体变量来满足我们的需求，所以我们就需要一个动态的链表，我们需要多少它就给多少。

#include <stdio.h>
#include <malloc.h>
struct student
{
    int a;
    struct student *next; // 定义结构体指用来指向下一个结构体的地址
};
struct student *fun(int a) // 定义结构体指针函数用来动态定义链表并返回链表的头指针
{
    int q = 0;
    struct student *new, *head, *last;                       // head指针指向链表的头指针，new指针指向新创建的结构体地址，last指针指向链表最后一个结构体
    head = (struct student *)malloc(sizeof(struct student)); // 先给头指针分配内存空间，不分配默认为4个字节
    last = head;                                             // 尾指针指向链表的最后一个结构体的地址
    head->a = q;
    while (a--)
    {
        q++;
        new = (struct student *)malloc(sizeof(struct student)); // 给新添加的结点分配内存空间
        new->a = q;
        last->next = new; // 链表与新创建的结构体链接起来
        last = new;       // 尾指针指向链表的最后一个结构体地址
    }
    last->next = NULL; // 尾指针的值为空代表链表结束
    return head;       // 返回链表的头节点
}
void main()
{
    int c;
    struct student *p; // 用来接收链表的头节点
    scanf("%d", &c);
    p = fun(c);
    while (p->next != NULL)
    {
        printf("%d", p->a);
        p = p->next;
    } // 循环遍历链表并输出值
}

输入5

输出的结果为01234

注意上述malloc分配的地址空间是连续的，比如为int *p整型指针分配8个字节，则8个字节的内存空间是连续的，并且可以用 *(p+0)和 *(p+1)来分别为其赋值。

关键字的介绍

break

break用于打断循环，也就是执行到break，直接跳出循环，可以打断for，switch，while，do..while。break可以跳出for，但是如果用一个以上的for，则是打断离它最近的for。

continue

continue用于跳出本次循环，不是退出循环，而是只跳出本次的循环

static

1.修饰全局变量，如果static修饰了全局变量，那只能在本工程使用，其他工程就不能调用了，

2.修饰局部变量，它是存储在静态存储区的，用static修饰过后，就是函数执行结束，值依然在，如果static未被赋值，默认值就是 0；

3.static修饰函数，函数也只能在本工程使用，其他工程不可调用。

一.链表的反转

#include <stdio.h>
struct ListNode
{
    struct ListNode *next;
    int a;
} a, b, c, *p, *head;
struct ListNode *ReverseList(struct ListNode *pHead)//反转的指针函数
{
    // write code here
    struct ListNode *p, *q;
    if (pHead != NULL)
    {
        p = NULL;
        while (pHead->next != NULL)
        {
            q = p;
            p = pHead;
            pHead = pHead->next;
            p->next = q;
        }
        pHead->next = p;//遍历完之后要把头结点指向p
    }
    return pHead;
}
void main()
{
    a.a = 1;
    b.a = 2;
    c.a = 3;
    head = &a;
    a.next = &b;
    b.next = &c;
    c.next = NULL;
    p = ReverseList(head);
    while (p != NULL)
    {
        printf("%d", p->a);
        p = p->next;
    }
}

本题用了两个指针，q和p分别指向头结点的前两个结点，可以说头节点带着q和p去找链表的结点，p改变结点的指向，

q保存了被指向的结点。

输出结果为321

二.链表的合并

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */


struct ListNode* mergeTwoLists(struct ListNode* list1, struct ListNode* list2){
    if(list1==NULL)
    {
        return list2;
    }
    else if(list2==NULL)
    {
        return list1;
    }
    //前面两个条件剔除了空链表的情况
    else if(list1->val<list2->val)
    {
        list1->next= mergeTwoLists(list1->next,list2);//让最小的节点指向下一次遍历返回的结果
        return list1;//返回较小的节点
    }
    else 
    {
        list2->next=mergeTwoLists(list1,list2->next);
        return list2;
    }
}

本题用到了递归算法，每次把两个链表较小的那个结点放入栈 的最低端，一直到其中一个链表遍历完，如何释放，得到链表的头结点点。
